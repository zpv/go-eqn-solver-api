    Introduction

    Welcome to the Golang Equation Solver API! You can use our API to solve any linear equation with one unknown variable.

    Demo the API here.

    Solver

    Solve Equation

    Single solution

    curl https://go-eqn-solver.azurewebsites.net/api/solve \
  --header "Content-Type: application/json" \
  --request POST \
  --data '{"eqn":"2x=4"}'

    {
  "result": "x=2",
}

    Infinite solutions

    curl https://go-eqn-solver.azurewebsites.net/api/solve \
  --header "Content-Type: application/json" \
  --request POST \
  --data '{"eqn":"2x=2x"}'

    {
  "result": "Infinite solutions",
}

    No solution

    curl https://go-eqn-solver.azurewebsites.net/api/solve \
  --header "Content-Type: application/json" \
  --request POST \
  --data '{"eqn":"x+5=x"}'

    {
  "result": "No solution",
}

    This endpoint solves a given linear equation.

    HTTP Request

    POST https://go-eqn-solver.azurewebsites.net/api/solve

    JSON Request

    Key Description Example
    eqn A linear equation with one unknown x 2x=4

    JSON Response

    Given a valid request, here are the possible responses. The response is returned as JSON with a single result field.

    Result Description
    x={number} The equation has a single solution, number, rounded to 8 decimal places.
    Infinite solutions The equation has infinite solutions
    No solution The equation is unsolvable

    Errors

    The Solver API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request is invalid. The linear equation may be malformed.
